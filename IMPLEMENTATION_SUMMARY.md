# Реализация разделов физической информации

## Выполненные задачи

Добавлены следующие разделы физической информации в файл `src/lib/physics-info.ts`:

1. **Первый закон термодинамики** (`firstLawThermodynamics`)
   - Закон сохранения энергии: ΔU = Q - W
   - Приложения: тепловые двигатели, холодильники, термостаты, биологические системы

2. **Второй закон термодинамики** (`secondLawThermodynamics`)
   - Закон увеличения энтропии: ΔS ≥ 0
   - Приложения: тепловые двигатели, холодильники, химические реакции, теплопередача

3. **Третий закон термодинамики** (`thirdLawThermodynamics`)
   - Энтропия при абсолютном нуле: S → 0 при T → 0 K
   - Приложения: криогенные системы, сверхпроводимость, точные измерения энтропии

4. **Электропроводимость газов** (`gasConductivity`)
   - Способность газа проводить электрический ток при ионизации
   - Формула: j = σE
   - Приложения: люминесцентные лампы, неоновые вывески, плазменные панели

5. **Принцип Ферми** (`fermiPrinciple`)
   - Принцип запрета Паули для фермионов
   - Формула: распределение Ферми-Дирака
   - Приложения: электронная проводимость в металлах, полупроводники, белые карлики

6. **Закон Ома** (`ohmLaw`)
   - Линейная зависимость тока от напряжения: I = U/R
   - Приложения: электрические цепи, нагревательные элементы, датчики

7. **Электричество** (`electricity`)
   - Общий раздел об электрических явлениях
   - Формула: закон Кулона F = k·q₁·q₂/r²
   - Приложения: энергетика, электроника, связь, транспорт

8. **Робототехника** (`robotics`)
   - Отрасль техники по проектированию и применению роботов
   - Приложения: промышленная автоматизация, хирургические роботы, космические роботы

9. **Схемотехника** (`schematics`)
   - Проектирование электрических и электронных схем
   - Приложения: проектирование печатных плат, интегральные схемы, микроконтроллеры
