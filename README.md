## 🚀 Быстрый старт

### Требования

| Компонент | Версия             | Ссылка                              |
| --------- | ------------------ | ----------------------------------- |
| Node.js   | 18+                | [nodejs.org](https://nodejs.org/)   |
| Bun       | 1.0+ (опционально) | [bun.sh](https://bun.sh/)           |
| Git       | Любая              | [git-scm.com](https://git-scm.com/) |

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/QuadDarv1ne/quantum-horizon.git
cd quantum-horizon

# Установка зависимостей
npm install
# или с bun
bun install

# Копирование переменных окружения
cp .env.example .env.local

# Запуск разработки
npm run dev
# или
bun run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 📦 Сборка и запуск

```bash
# Production сборка
npm run build
# или
bun run build

# Запуск production сервера
npm run start
# или
bun run start
```

### Docker (опционально)

```bash
# Сборка образа
docker build -t quantum-horizon .

# Запуск контейнера
docker run -p 3000:3000 quantum-horizon
```
