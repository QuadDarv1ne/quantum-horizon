# ⧫ QUANTUM HORIZON

> **От частиц до космоса • From Particles to Cosmos**

Интерактивная платформа для визуализации и изучения современной физики — от квантовой механики до космологии.

## 🌌 Возможности

### Квантовая механика
- Волновая функция (уравнение Шрёдингера)
- Принцип неопределённости Гейзенберга
- Квантовое туннелирование
- Эксперимент с двойной щелью
- Кот Шрёдингера
- Квантовая запутанность

### Теория относительности
- Замедление времени
- Сокращение длины
- E = mc²

### Космология
- Диаграмма Герцшпрунга-Рассела
- Нейтронные звёзды и пульсары
- Чёрные дыры с излучением Хокинга
- Тёмная материя
- Большой взрыв
- Реликтовое излучение

### Продвинутые темы
- Фотоэффект
- Броуновское движение
- Гравитационные волны
- Атомная модель Бора
- Радиоактивный распад
- Суперпроводимость
- Стандартная модель

### Дополнительно
- Калькулятор физических формул
- Таймлайн открытий
- Солнечная система
- Тест по физике
- Биографии учёных

### Языки
🇷🇺 Русский • 🇬🇧 English • 🇨🇳 中文 • 🇮🇱 עברית

## 🚀 Установка

```bash
git clone https://github.com/QuadDarv1ne/quantum-horizon.git
cd quantum-horizon
npm install
npm run dev
```

Откройте http://localhost:3000

---

## 📦 Команды

```bash
npm run dev      # Разработка
npm run build    # Production сборка
npm run start    # Production сервер
npm run lint     # Проверка кода
npm run format   # Форматирование
```

### Docker

```bash
docker build -t quantum-horizon .
docker run -p 3000:3000 quantum-horizon
```
