- Работаю в PowerShell (Windows)
- Git workflow: сначала коммитим изменения в ветку dev, затем вливаем dev в main через merge/pull request
- Git commit в PowerShell: использовать команду `echo "сообщение" > .git/COMMIT_EDITMSG && git commit -F .git/COMMIT_EDITMSG`. НЕ использовать кавычки напрямую в -m или --message (PowerShell разбивает на аргументы)
